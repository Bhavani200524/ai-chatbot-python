# ai-chatbot-python
A simple AI Chatbot developed using Python and Tkinter that interacts with users through a graphical user interface and responds to predefined questions.

## Overview
This project is a simple AI Chatbot developed using Python and Tkinter. The chatbot interacts with users and responds to predefined questions through a graphical user interface.

## Features
- User-friendly GUI
- Chat-based interaction
- Predefined AI responses
- Simple Python implementation
- Beginner-friendly project

## Technologies Used
- Python
- Tkinter

## Working Principle
The chatbot accepts user input through a text box and provides responses based on predefined conditions written in Python.
